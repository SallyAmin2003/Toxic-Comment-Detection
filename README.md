# Deep Learning-Based Toxic Comment Detection

## Overview

This project uses Natural Language Processing (NLP) and Deep Learning to detect toxic comments in online communities.

## Dataset

Jigsaw Toxic Comment Classification Dataset

## Technologies

- Python
- TensorFlow
- Keras
- Scikit-learn
- Pandas
- NumPy

## Model

- Bidirectional LSTM (BiLSTM)
- Binary Classification
- Multi-label Classification

## Results

- Accuracy: 92.81%
- ROC-AUC: 0.9685
- Average Precision: 0.8628

## Future Improvements

- Transformer Models (BERT)
- Better handling of rare labels
- Real-time moderation API
